Una volta scansionato con successo il QR code, il sistema verifica la validità del codice, "brucia" il QR code in modo che non sia più utilizzabile, ed eroga istantaneamente il cashback sul conto Postepay del cliente. Viene mostrata una schermata di conferma con tutti i dettagli della transazione.

Riepilogo del processo

1. Dashboard

Il commerciante accede all'app e visualizza la dashboard con le statistiche di cashback e le ultime transazioni.

2. Importo

Inserisce l'importo dell'acquisto effettuato dal cliente e il sistema calcola automaticamente il cashback da erogare.

3. Scansione

Scansiona il QR code generato dal cliente nell'app P, che contiene le informazioni necessarie per l'accredito.

4. Conferma

Il sistema conferma l'avvenuta erogazione del cashback sul conto del cliente con tutti i dettagli della transazione.

Aspetti tecnici e di sicurezza

Come funziona il QR code

Il QR code generato nell'app cliente contiene:

  • Identificativo univoco della carta Postepay del cliente
  • Token di sicurezza con validità temporale limitata (24 ore)
  • Identificativo del punto vendita a cui è associata la promozione
  • Percentuale di cashback associata alla promozione

Elaborazione dell'operazione

Quando l'app ScontiPosteBusiness scansiona il QR code:

  1. Verifica la validità del token e l'identità del cliente
  2. Calcola l'importo del cashback in base alla percentuale prevista
  3. Invia una richiesta di accredito al sistema di pagamento Postepay
  4. Riceve la conferma dell'avvenuto accredito
  5. Marca il QR code come "utilizzato" nel database
